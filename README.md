##  Discount Allocation System

Distributes a discount kitty across agents based on:
- Performance
- Seniority
- Target Achievement
- Active Clients

##  Run Instructions

```bash
npm install
npm start
```

## Run Tests

```bash
npm test
```

##  Justification Categories

- ≥85: High performer
- 60–84: Moderate performer
- <60: Low performer

##  Extensible
- Easily configurable weights
- Add CLI or API layer on top
